---
title: Helper Scripts
---

- Scaffold a complete API with its configurations, controller, model and service.

  ```bash
  yarn api <name>

  #  example
  yarn api order
  ```

- new schema template

  ```bash
  yarn schema <name>

  # example
  yarn schema createProduct
  ```

- new schema policy

  ```bash
  yarn policy <name>

  # example
  yarn policy isCreateProduct
  ```
